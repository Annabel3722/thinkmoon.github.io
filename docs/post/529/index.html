复习一个计组知识（cache“读不命中”的解决办法）

  1. 将内存中的数据复制到Cache中，然后把这个字传给CPU。
  2. 启动常规的主存读取周期，将字从主存中读出在并送到CPU，同时将这个字送入Cache。

原理类比

软件请求 => CPU运算
云对象存储（CDN分发)响应 => 高速缓存Cache读取
服务器带宽响应 => 主存读取
